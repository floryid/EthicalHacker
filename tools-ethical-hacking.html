üîç Reconnaissance & Scanning Tools
Nmap (Network Mapper)
Reconnaissance, Port Scanning, Network Enumeration
Nmap adalah tool open-source paling populer untuk scanning dan mapping jaringan. Digunakan untuk mendeteksi hosts, ports, services, dan vulnerabilities.

Instalasi

Linux: apt install nmap
Windows: Download dari nmap.org
macOS: brew install nmap

Kegunaan Utama

Port scanning, OS detection, service enumeration, vulnerability scanning

Cheat Sheet - Command Dasar:

Basic Port Scan
nmap 192.168.1.1
Scan Specific Ports
nmap -p 80,443,22 192.168.1.1
Aggressive Scan
nmap -A -T4 192.168.1.1
OS Fingerprinting
nmap -O 192.168.1.1
Service Version Detection
nmap -sV 192.168.1.1
UDP Scan
nmap -sU 192.168.1.1
Stealth Scan (SYN Scan)
nmap -sS -Pn 192.168.1.1
Scan Entire Network
nmap 192.168.1.0/24
Penjelasan: Port scanning adalah tahap pertama dalam penetration testing. Dengan Nmap Anda dapat menemukan port mana yang terbuka, layanan apa yang berjalan, dan versi software apa yang digunakan.

Fitur Unggulan Nmap:

  • Port scanning (TCP, UDP)
  • OS dan service detection
  • Scripting engine (NSE)
  • Firewall/IDS evasion
  • Network mapping
Shodan
Internet Search Engine, Reconnaissance, OSINT
Shodan adalah search engine yang mengindex device yang terhubung ke internet. Digunakan untuk menemukan IoT devices, webcams, servers yang exposed.

Shodan Query Examples:

Cari Webcam
webcam country:"US"
Cari Jenkins Server
jenkins
Cari Router
router type:"routing"
Cari Default Credentials
http.title:"Apache" port:80
‚ö†Ô∏è Warning: Shodan hanya untuk reconnaissance. Jangan scan/exploit device tanpa authorization!
TheHarvester
OSINT, Email Harvesting, Information Gathering
Tool untuk gathering email addresses, subdomains, dan hosts menggunakan berbagai public sources seperti Google, Bing, LinkedIn, etc.

Cheat Sheet - Command Dasar:

Harvest Email dari Google
theHarvester -d example.com -l 100 -b google
Harvest dari Multiple Sources
theHarvester -d example.com -l 200 -b all
Output ke HTML Report
theHarvester -d example.com -b google -f report.html
üì° Network Analysis & Web Testing Tools
Wireshark
Network Sniffer, Packet Analysis, Protocol Analyzer
Wireshark adalah tool GUI untuk capturing dan analyzing network traffic secara real-time. Essential untuk troubleshooting jaringan dan security analysis.

Penggunaan Dasar:

Langkah-langkah:
  1. Buka Wireshark
  2. Pilih network interface yang ingin di-capture
  3. Klik tombol Start untuk mulai capturing packets
  4. Perform network activities (browse website, streaming, etc)
  5. Analisis packets yang tertangkap

Filter Expressions:

Filter Traffic Specific IP
ip.addr == 192.168.1.100
Filter HTTP Traffic
http
Filter HTTPS/TLS Traffic
tls
Filter TCP Port 22 (SSH)
tcp.port == 22
Filter DNS Queries
dns.qry.name contains example.com
Filter Password Attempts
http.request.method == POST

Kapabilitas Wireshark:

  • Live packet capture dan offline analysis
  • Deep packet inspection
  • Powerful filtering & searching
  • Export packets dalam berbagai format
  • Protocol decoding
Burp Suite
Web Application Security Testing, Proxy, Scanner
Burp Suite adalah platform terlengkap untuk web application penetration testing. Mengintegrasikan proxy, scanner, repeater, dan berbagai tools lainnya.

Versi

Community (Free) & Professional (Paid)

Fitur Utama

Web proxy, Scanner, Repeater, Intruder, Decoder

Workflow Dasar Burp Suite:

Langkah-langkah:
  1. Configure browser proxy ke Burp (port 8080 default)
  2. Navigate website melalui Burp proxy
  3. Requests akan tertangkap di Proxy tab
  4. Review requests di Target site map
  5. Gunakan Repeater untuk testing manual
  6. Jalankan Scanner untuk automated testing

Fitur Penting:

Intruder - Brute Force Attack
1. Capture request di Proxy 2. Send to Intruder 3. Tentukan payload positions 4. Load password list 5. Start attack

Burp Suite Tools:

  • Proxy - Intercept requests/responses
  • Scanner - Find vulnerabilities
  • Repeater - Manual request testing
  • Intruder - Brute force & fuzzing
  • Decoder - Encoding/decoding
  • Comparer - Compare data
‚ö†Ô∏è Important: Hanya gunakan pada aplikasi yang Anda miliki atau memiliki explicit permission!
üí• Exploitation & Vulnerability Tools
Metasploit Framework
Exploitation, Penetration Testing, Post-Exploitation
Metasploit adalah framework paling powerful untuk penetration testing. Berisi ribuan exploits, payloads, dan auxiliary modules untuk berbagai vulnerabilities.

Basic Metasploit Workflow:

Start Metasploit Console
msfconsole
Search untuk Module
search smb
Gunakan Module
use exploit/windows/smb/ms17_010_eternalblue
Set Required Options
set RHOST 192.168.1.100 set LHOST 192.168.1.50 set LPORT 4444 show options
Run Exploit
exploit
Metasploit Terminology:
- Exploit: Code yang memanfaatkan vulnerability
- Payload: Code yang dijalankan setelah exploit berhasil
- Shellcode: Machine code yang executed pada target
- Listener: Mendengarkan reverse shell dari target

Popular Metasploit Modules:

EternalBlue (MS17-010)
exploit/windows/smb/ms17_010_eternalblue
Apache Tomcat Upload
exploit/windows/http/tomcat_cgi_cmdexec
SSH Brute Force
auxiliary/scanner/ssh/ssh_login
SMB Share Enumeration
auxiliary/scanner/smb/smb_enumshares

Metasploit Capabilities:

  • 1000+ exploits dan payloads
  • Vulnerability scanning
  • Post-exploitation modules
  • Meterpreter sessions management
  • Social engineering toolkit integration
SQLMap
SQL Injection Testing, Web Exploitation
SQLMap adalah tool automated untuk detecting dan exploiting SQL injection vulnerabilities. Dapat extract database, bypass authentication, dan full system takeover.

SQLMap Cheat Sheet:

Basic SQL Injection Test
sqlmap -u "http://target.com/page.php?id=1" --dbs
List All Databases
sqlmap -u "http://target.com/page.php?id=1" --dbs
List Tables in Database
sqlmap -u "http://target.com/page.php?id=1" -D database_name --tables
Extract Columns from Table
sqlmap -u "http://target.com/page.php?id=1" -D database_name -T users --columns
Dump Table Data
sqlmap -u "http://target.com/page.php?id=1" -D database_name -T users --dump
From Burp Request
sqlmap -r request.txt --dbs
POST Method Testing
sqlmap -u "http://target.com/login.php" --data="username=admin&password=123" --dbs
‚ö†Ô∏è Legal Warning: SQL injection attacks tanpa authorization adalah ILLEGAL! Hanya gunakan untuk testing systems yang Anda punya izin!
Hydra
Password Cracking, Brute Force, Credential Testing
Hydra adalah fast network login cracker yang support 50+ protocols. Digunakan untuk brute force password pada SSH, FTP, HTTP, RDP, dan lainnya.

Hydra Command Examples:

SSH Brute Force
hydra -l username -P passwords.txt ssh://192.168.1.100
Multiple Users
hydra -L users.txt -P passwords.txt ssh://192.168.1.100
FTP Brute Force
hydra -l admin -P passwords.txt ftp://192.168.1.100
HTTP Basic Auth
hydra -l admin -P passwords.txt http-basic://192.168.1.100
RDP (Windows Remote Desktop)
hydra -L users.txt -P passwords.txt rdp://192.168.1.100
Increase Threads (Faster)
hydra -l admin -P passwords.txt -t 16 ssh://192.168.1.100
Save Output
hydra -l admin -P passwords.txt -o results.txt ssh://192.168.1.100

Hydra Supported Protocols:

  • SSH - Secure Shell
  • FTP - File Transfer
  • HTTP/HTTPS - Web
  • Telnet - Remote login
  • RDP - Windows Remote Desktop
  • MySQL, PostgreSQL, Oracle
  • SNMP - Network monitoring
üîê Password Cracking & Hash Tools
John the Ripper
Password Cracking, Hash Cracking, Dictionary Attack
John the Ripper adalah tool powerful untuk offline password cracking. Support berbagai hash types dan password cracking methods.

John the Ripper Cheat Sheet:

Basic Crack dari Wordlist
john --wordlist=wordlist.txt hashes.txt
Identify Hash Type
john --format=dynamic=17 hashes.txt
Common Hash Types
john --format=md5 hashes.txt john --format=sha1 hashes.txt john --format=bcrypt hashes.txt
Show Found Passwords
john --show hashes.txt
Increment Mode (Brute Force)
john --mode=incremental:alpha hashes.txt
Custom Rules
john --wordlist=wordlist.txt --rules hashes.txt
Extract Hashes from Shadow File
unshadow /etc/passwd /etc/shadow > hashes.txt
Hashcat
GPU-Accelerated Password Cracking, Hash Cracking
Hashcat adalah tool password cracking tercepat yang menggunakan GPU acceleration. Support 300+ hash types dan berbagai attack modes.

Hashcat Attack Modes:

Dictionary Attack (Mode 0)
hashcat -m 0 hashes.txt wordlist.txt
Brute Force (Mode 3)
hashcat -m 0 -a 3 hashes.txt ?a?a?a?a
Combination Attack (Mode 1)
hashcat -m 0 -a 1 hashes.txt dict1.txt dict2.txt
Mask Attack
hashcat -m 0 -a 3 hashes.txt ?u?l?l?l?d?d
List Available Hash Types
hashcat --help | grep "Hash type"

Hashcat Features:

  • GPU acceleration (100x faster)
  • 300+ hash types supported
  • Multiple attack modes
  • Rule-based cracking
  • Session recovery
üñ•Ô∏è Reverse Engineering Tools
Ghidra
Reverse Engineering, Disassembly, Decompilation
Ghidra adalah free reverse engineering framework dari NSA. Powerful untuk disassembly, decompilation, dan binary analysis.

Ghidra Basics:

Langkah-langkah Menggunakan Ghidra:
  1. Buka Ghidra Project Manager
  2. Create new project atau open existing
  3. Import binary file (EXE, DLL, ELF, dll)
  4. Ghidra akan auto-analyze binary
  5. Explore fungsi di Listing window
  6. View decompiled code di Decompiler window
  7. Rename functions dan variables

Ghidra Shortcuts:

Search String dalam Binary
Ctrl+F (Search String)
Jump to Address
Ctrl+G (Go To Address)
Find Cross-References
Ctrl+Shift+F (Find References)
Show Function Calls
Ctrl+B (Show Call Tree)

Ghidra Capabilities:

  • Support multiple architectures (x86, ARM, MIPS, dll)
  • Powerful decompiler
  • Graph-based code visualization
  • Script automation
  • Collaboration features
GDB (GNU Debugger)
Debugging, Binary Analysis, Dynamic Analysis
GDB adalah standard debugger untuk Linux. Digunakan untuk dynamic analysis, debugging, dan exploit development.

GDB Cheat Sheet:

Start GDB dengan Binary
gdb ./program
Set Breakpoint
break main break *0x08048000 b function_name
Run Program
run run arg1 arg2
Continue Execution
continue (c)
Step Into / Step Over
step (s) - Step Into next (n) - Step Over
Print Variable Value
print variable_name print *pointer x/4x $esp
Show Disassembly
disassemble main disas *$pc
Show Stack
info stack bt (backtrace)

Password Cracking Tool Comparison:

John the Ripper

  • CPU-based cracking
  • Good rule support
  • Slower than GPU tools
  • Cross-platform
  • Open-source

Hashcat

  • GPU-accelerated (100x faster)
  • More hash types
  • Better performance
  • Requires good GPU
  • Open-source

RainbowCrack

  • Lookup-table based
  • Extremely fast
  • Pre-computed tables required
  • Limited flexibility
  • Good for common hashes

Web Testing Tool Comparison:

Burp Suite Community

  • Free version tersedia
  • Manual testing tools
  • Good proxy & repeater
  • Limited scanner
  • Excellent for learning

Burp Suite Pro

  • Full automation scanner
  • Enterprise features
  • Mahal (premium pricing)
  • Industry standard
  • Best for professionals

OWASP ZAP

  • 100% open-source & free
  • Good automation
  • Less polished UI
  • Growing community
  • Great alternative

‚úÖ Best Practices & Safety

‚öñÔ∏è Legal & Ethical Guidelines:

  • Authorization: Always get written permission sebelum testing sistem orang lain
  • Scope: Hanya test systems dan services yang included dalam scope
  • Responsible Disclosure: Report findings responsibly dan rahasia
  • No Data Theft: Jangan extract atau steal data unnecessary
  • No Disruption: Hindari actions yang disrupt normal operation
  • Documentation: Document semua activities untuk compliance

Tool Safety Practices:

  • Isolated Environment: Test di lab environment, bukan production
  • Time Windows: Performa testing hanya di agreed time windows
  • Backup: Backup semua systems sebelum penetration testing
  • Monitoring: Monitor systems during testing untuk immediate detection
  • Clean-up: Remove semua test artifacts setelah testing selesai
  • Report: Provide detailed report dengan findings dan recommendations
